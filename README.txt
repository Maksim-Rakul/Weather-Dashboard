🌦 Weather Dashboard

📌 Description

Weather Dashboard is a web application that allows users to check the current weather and a 5-day forecast.

The app provides weather data by city name or user geolocation, with additional customization features for a better user experience.

✨ Features
🔍 Search weather by city name
📍 Get weather using geolocation 
🌗 Toggle between light and dark themes
🌡 Switch between Celsius and Fahrenheit
📅 View a 5-day weather forecast


🎯 Purpose

This project was built as a pet project for portfolio purposes to demonstrate practical frontend development skills and working with APIs.

🛠 Tech Stack

Languages: HTML, CSS, JavaScript
Build Tool: Vite
API: OpenWeather API
Libraries:
Axios — for HTTP requests
IziToast — for notifications
🚀 Deployment

The project is deployed using Vercel.
